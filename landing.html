Ti-Nspire CX

Exploring the connection between computer technology and the design process

Illustrator

This is an orthographic drawing of a Ti-Nspire CX calculator. This was drawn in Illustrator using tools such as the pen, line, shape, duplication, join, and layers. Reference views were used to create an accurate 1:1 scale view of the calculator

Photoshop

This was rendered in Photosop using tools such as the paintbrush, layers, mask, and blend modes. A non destructive shading technique was used to create this rendereing.

Axure

This interface was created with Axure using tools such as dynamic panels and sequenes. The interaction was based off the real device and the buttons highlighted in blue are used in the interface. View prototype at: http://v2ytc7.axshare.com

After Effects

This was created in AfterEffects using tools such as overlays transformation tools, and animation effects. The Photoshop rendering was used with the Axure interface placed ontop of it.
